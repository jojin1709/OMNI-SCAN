All-in-One Bug Bounty & Pentesting Scanner

v1.0.0 Python Security

What is OmniScan?

OmniScan is an all-in-one web application security scanner designed for bug bounty hunters and penetration testers. It detects 18 vulnerability classes in a single unified scan, including SQL Injection, XSS, RCE, IDOR, Privilege Escalation, and more.

🎯 Features

40+ Detectors

Run all vulnerability detectors in parallel for comprehensive coverage

Smart Crawling

Discovers endpoints, forms, API routes, JS files, and hidden parameters

Multi-Format Reports

JSON, HTML, Markdown, and TXT output formats

Authentication Support

Form login, cookies, JWT, OAuth, and bearer tokens

OOB Testing

Built-in callback server for blind vulnerabilities

Proxy & Tor

Support for Burp, ZAP, and Tor routing

📦 Installation

# Clone the repository
git clone https://github.com/jojin1709/OMNI-SCAN.git
cd OMNI-SCAN

# Install dependencies
pip3 install -r requirements.txt

🚀 Quick Start

Basic Scan

python3 omniscan.py -t https://target.com --all

Quick Scan (High-Signal Only)

python3 omniscan.py -t https://target.com --quick

With Authentication

python3 omniscan.py -t https://target.com \
    --cookie "sessionid=abc123" \
    --all

With Callback Server

python3 omniscan.py -t https://target.com \
    --callback-port 8888 \
    --all
